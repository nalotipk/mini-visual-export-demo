# Backend (NestJS)

This folder contains the NestJS backend for the Mini Visual Builder project.

- Endpoint: `POST /export/zip`
- Generates a complete React + Tailwind project
- Writes files to a temp directory
- Bundles everything using `archiver`
- Streams ZIP back to the frontend
